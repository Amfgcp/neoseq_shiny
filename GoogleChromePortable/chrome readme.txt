Install chrome portable in this folder.



app.R references an executable in this directory (GoogleChromePortable\GoogleChromePortable.exe). 
If you are using a different version of Chrome portable, make sure the least nestled .exe is referenced.

After installing, run and install extension "No MouseWheel Zoom" (https://chrome.google.com/webstore/detail/no-mousewheel-zoom/ckhlfagjncfmdieecfekjbpnnaekhlhd/reviews?hl=en-US).



Tested with GoogleChromePortable v.69.0.3497.100.
